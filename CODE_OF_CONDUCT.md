# Code of Conduct

This project follows the [Contributor Covenant][cc], version 2.1.

The full text, including the standards expected of contributors,
the responsibilities of maintainers, and the enforcement steps,
is available verbatim on the Contributor Covenant website:

  <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

GitHub recognises this file as the project's Code of Conduct and
surfaces it via the repository's *Community Standards* tab.

## Reporting concerns

If you believe someone has violated the Code of Conduct in a
project space (issues, pull requests, discussions, commit messages,
or any other channel directly tied to this repository), please
contact the maintainer privately at the address listed in the
project's [Impressum / contact information](README.md).

Reports are read by the maintainer only and treated confidentially.
You do not need to use any particular wording or template. A short
description of what happened, where it happened, and (if relevant)
links to the messages in question is enough.

## Enforcement

The maintainer commits to enforcing the Contributor Covenant fairly
and consistently across all contributors, regardless of their
status, history with the project, or relationship to the maintainer.

The maintainer reserves the right to remove comments, commits, code,
issues, pull requests, and other contributions that are not aligned
with this Code of Conduct, and to temporarily or permanently ban any
contributor for behaviours they deem inappropriate, threatening,
offensive, or harmful, following the enforcement guidelines linked
above.

## Scope

This Code of Conduct applies within all project spaces, and also
applies when an individual is officially representing the project
in public spaces.
