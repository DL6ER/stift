# Keyboard Shortcuts

Press `?` inside the app for an in-context reference panel.

## Tools

| Key | Tool |
|-----|------|
| `V` | Select / Move |
| `A` | Arrow |
| `T` | Text |
| `G` | Text Box |
| `H` | Highlight |
| `B` | Blur |
| `R` | Rectangle |
| `E` | Ellipse |
| `L` | Line |
| `D` | Freehand Draw |
| `X` | Color Box (redact) |
| `N` | Counter |
| `M` | Dimension / Measure |
| `W` | Stamp / Watermark |
| `K` | Connector |

## Editing

| Key | Action |
|-----|--------|
| `Ctrl+Z` / `Ctrl+Y` | Undo / Redo |
| `Ctrl+C` / `Ctrl+V` | Copy / Paste |
| `Ctrl+D` | Duplicate |
| `Ctrl+A` | Select all |
| `Ctrl+S` | Save locally |
| `Delete` | Remove selected |
| `Escape` | Deselect |
| `Arrow keys` | Nudge 1px (`Shift`: 10px) |
| `]` / `[` | Layer forward / backward |
| `Shift+]` / `Shift+[` | Layer to front / back |
| `Shift+click` | Multi-select |

## Canvas

| Key | Action |
|-----|--------|
| `Space+drag` | Pan canvas |
| `Scroll` | Zoom |
| `?` | Open shortcut reference |
